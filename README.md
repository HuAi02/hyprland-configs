## Dependencies

hyprpaper (wallpaper)\
hypridle (idle management)\
hyprlock (lockscreen)\
wlogout (power menu)\
nm-applet (WiFi tray icon)\
blueman-applet (Bluetooth tray icon)\
brightnessctl (brightness control)\
wpctl (volume control)\
auto-cpufreq & powertop (power management)\
fcitx5 (pinyin input support)
